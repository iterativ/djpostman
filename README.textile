h1. DjPostman

Advanced e-Mail handler

Features:
* Asynchronous email sending (via celery)
* Store sent email in database (match auth.User)
* Recevice and store email from a mail account (via imap, match auth.User)
* Send mail inside Django Admin
* Easy to use multi alternatives email rendering
